<tt>ctime</tt> - C Time Library

Data

  • time_t - calendar time, usually seconds since Unix epoch
  • clock_t - CPU time ticks returned by clock()
  • struct tm - broken-down calendar time
  • struct timespec - seconds + nanoseconds
  • struct timeval - seconds + microseconds (Unix)

Constants

  • CLOCKS_PER_SEC - number of clock_t ticks per second
  • TIME_UTC - base for timespec_get()

Manipulation

  • clock() - get processor time used by program
  • time() - get current calendar time
  • timespec_get() - get current time as struct timespec (C11/C++17)
  • timespec_getres() - get resolution for timespec_get() clock (C23)
  • difftime() - return difference between two times

Conversion <tt>struct tm</tt> to

  • mktime() - convert to time_t
  • asctime() - convert to string
  • strftime() - format as string

Conversion <tt>struct tm</tt> to (Unix)

  • timegm() - convert UTC tm to time_t
  • timelocal() - convert local tm to time_t, same as mktime()

Conversion <tt>time_t</tt> to

  • ctime() - convert to string
  • gmtime() - convert to tm as UTC time
  • localtime() - convert to tm as local time

Thread-safe conversion (POSIX)

  • asctime_r() - convert tm to caller-provided string buffer
  • ctime_r() - convert time_t to caller-provided string buffer
  • gmtime_r() - convert time_t to caller-provided UTC tm
  • localtime_r() - convert time_t to caller-provided local tm

Parsing (POSIX)

  • strptime() - parse string to struct tm

Clocks (POSIX)

  • clock_gettime() - get time from selected clock
  • clock_getres() - get selected clock resolution
  • clock_settime() - set selected clock time

Timezone (POSIX)

  • tzset() - initialize timezone data from environment

<tt>struct timeval</tt> macros (Unix)

  • timerisset() - check if value is non-zero
  • timerclear() - reset value to zero
  • timercmp() - compare two values
  • timeradd() - add two values
  • timersub() - subtract two values
